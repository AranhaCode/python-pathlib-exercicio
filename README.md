# python-pathlib-exercicio
Exercício prático de automação com Python utilizando a biblioteca pathlib. O script gerencia a criação de diretórios e arquivos de forma inteligente com match-case
Este pequeno script faz parte dos meus estudos de lógica de programação voltados para **DevOps**. 

### O que ele faz:
- Verifica a existência de pastas usando `pathlib`.
- Utiliza a estrutura moderna `match-case` do Python 3.10+ para interação com o usuário.
- Cria diretórios (`mkdir`) e arquivos (`touch`) automaticamente.

### Tecnologias:
- Python 3.1x
- Library: `pathlib`
